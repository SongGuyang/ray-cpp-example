# Ray C++ Example
Show how to use ray C++ API to develop distribute applications.


# How to run the example

## dependency
- OS: macOS or Linux
- python 3.6+
- bazel 4.2
- gcc 9+

## run simple_kv_store
- git clone https://github.com/SongGuyang/ray-cpp-example.git
- cd ray-cpp-example/src/examples
- sh install_ray.sh
- sh run.sh
